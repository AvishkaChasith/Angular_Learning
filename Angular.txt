What is Angular?
    Angular is a front end framework build using the JavaScript by Google
    Angular is a front end JavaScript framework to build Client side applications using HTML, CSS, and JavaScript

What is SPA?
    A Single Page Application is an app that doesn't need to reload the page during its use and works within a Web Browser.

What is Angular CLI?
    Angular CLI is a Command Line Interface that use to 
                            Develop
                            Scaffold
                            Maintain
    An Angular Apllication Directly from a Command Shell

Install Angular 
    in command prompt - npm install -g @angular/cli

Generate new App
    ng new ProjectName

Generate new component
    ng generate component componentName 
    ng g c componentName

Run the program
    ng serve
